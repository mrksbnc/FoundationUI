# bo-button

A button is a control that allows the user to perform an action.

## Basic usage

```html
<bo-button :label="Hello World!" />
```

## Props

- `label` - The label of the button (optional)
  - either `label` or `prefixIcon` or `suffixIcon` must be provided
- `variant` - The variant of the button (optional)
  - default: `BoButtonVariant.primary`
- `shape` - The shape of the button (optional)
  - default: `BoButtonShape.default`
- `type` - The type of the button (optional)
  - default: `button`
- `disabled` - Whether the button is disabled (optional)
  - default: `false`
- `isLoading` - Whether the button is loading (optional)
  - default: `false`
- `loaderText` - The text to be displayed in the loader (optional)
  - default: `''`
- `size` - The size of the button (optional)
  - default: `BoSize.default`
- `prefixIcon` - The icon to be displayed before the label (optional)
  - default: `Icon.none`
  - either `label` or `prefixIcon` or `suffixIcon` must be provided
- `suffixIcon` - The icon to be displayed after the label (optional)
  - default: `Icon.none`
  - either `label` or `prefixIcon` or `suffixIcon` must be provided

## Disabled

The `disabled` prop can be used to disable the button. When the button is disabled, it will not respond to user interactions and will appear in a disabled state.

<Canvas of={BoButtonStories.Disabled} />

## Loading

The `isLoading` prop can be used to show a loading spinner. When the spinner is shown, the button will be disabled and will not respond to user interactions.

<Canvas of={BoButtonStories.isLoading} />

## Button variants

To change the color of the component, you can use the predifened `BoButtonVariant` enum. The `BoButtonVariant` enum includes the following variants:

- `primary`
- `secondary`
- `danger`
- `warning`
- `success`
- `dark`

<Canvas of={BoButtonStories.Variants} />

## Button shapes

To change the shape of the component, you can use the predifened `BoButtonShape` enum. The `BoButtonShape` enum includes the following shapes:

- `default`
- `pill`
- `outline`

<Canvas of={BoButtonStories.Shapes} />

## Button types

To change the type of the component, you can use the predifened `type` prop. The `type` prop should be a string that represents the type of the button. The `type` prop can take the values of the `button`, `submit`, and `reset`.

## Button sizes

As with any other component in the project the `bo-button` component also uses the `BoSize` enum to define the size of the button. The `BoSize` enum includes the following sizes:

- `extra-small`
- `small`
- `default`
- `large`
- `extra-large`

## With prefix icon

The `prefixIcon` prop can be used to display an icon before the label of the button. The `prefixIcon` prop should be a string that represents the name of the icon you want to display.

<Canvas of={BoButtonStories.WithPrefixIcon} />

## With suffix icon

The `suffixIcon` prop can be used to display an icon after the label of the button. The `suffixIcon` prop should be a string that represents the name of the icon you want to display.

<Canvas of={BoButtonStories.WithSuffixIcon} />

## Icon only button variant

To create an icon only button, you can use the `prefixIcon` prop without the `label` prop.
If the label is not provided, the padding and the general size of the button will be aligned
to match the icon correctly.
