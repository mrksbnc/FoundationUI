# bo-slot-modal

A slot modal is a component that can be used to display a modal dialog with any custom content
in the header, body and footer.

## Basic usage

The component to use is called `bo-slot-modal`.

```html
<bo-slot-modal
	:show-close-button="false"
	:bordered-footer="false"
	:bordered-header="false"
	@update:show=""
>
	<template #header> // </template>
	<template #body> // </template>
	<template #footer> // </template>
</bo-slot-modal>
```

## Props

- `show-close-button` - Whether to show the close button (optional)
  - default: `true`
- `bordered-header` - Whether to show a bordered header (optional)
  - default: `true`
- `bordered-footer` - Whether to show a bordered footer (optional)
  - default: `true`
- `width-in-px` - The width of the modal in pixels (optional)
- `width-in-percent` - The width of the modal in percent (optional)
- `width-as-tailwind-class` - The width of the modal as a tailwind class (optional)

## Slots

- `header` - The header slot of the modal
- `body` - The body slot of the modal
- `footer` - The footer slot of the modal

## Events

- `update:show` - The event is emitted on the `@click` event of the close button
  or when the `ESC` key is pressed
