# bo-alert-modal

An alert modal is a component that can be used to display an alert dialog. It supports
different variants (info, warning, error, success) and alignments (centered, default).

## Basic usage

The component to use is called `bo-alert-modal`.

```html
<bo-alert-modal
  :title="title"
  :content="content"
  :confirm-button-label="confirmButtonLabel"
  :confirm-button-variant="confirmButtonVariant"
  :cancel-button-label="cancelButtonLabel"
  :cancel-button-variant="cancelButtonVariant"
  @update:show="onCancel"
/>
```

## Props

- `title` - The title of the modal (optional)
  - default: `''`
- `content` - The content of the modal (optional)
  - default: `''`
- `confirm-button-label` - The label of the confirm button (optional)
  - default: `'Confirm'`
- `confirm-button-variant` - The variant of the confirm button (optional)
  - default: `BoButtonVariant.primary`
- `cancel-button-label` - The label of the cancel button (optional)
  - default: ``
- `cancel-button-variant` - The variant of the cancel button (optional)
  - default: `BoButtonVariant.secondary`
- `variant` - The variant of the modal (optional)
  - default: `BoAlertModalVariant.info`
- `alignment` - The alignment of the modal (optional)
  - default: `BoAlertModalAlignment.default`
- `withIcon` - Whether to show an icon in the header (optional)
  - default: `true` -`onSubmit` - The event to be emitted when the confirm button is clicked (optional)
  - default: `() => {}`
- `onCancel` - The event to be emitted when the cancel button is clicked (optional)
  - default: `() => {}`

## Variants

The `variant` prop can be used to change the variant of the modal. The `variant` prop should be a member of the `BoAlertModalVariant` enum.

- `info`
- `warning`
- `error`
- `success`

## Alignments

The `alignment` prop can be used to change the alignment of the modal. The `alignment` prop should be a member of the `BoAlertModalAlignment` enum.

- `centered`
- `default`

## With icon

The `withIcon` prop can be used to show or hide an icon in the header. The `withIcon` prop should be a boolean.
