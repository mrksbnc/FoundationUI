# bo-text

A label is a small piece of text that is used to display information.

## Basic usage

```html
<bo-text :text="Hello World!" />
```

## Props

- `text` - The text to be displayed (required)
- `size` - The size of the text (optional)
  - default: `BoFontSize.default`
- `weight` - The weight of the text (optional)
  - default: `BoFontWeight.regular`
- `fontFamily` - The font family of the text (optional)
  - default: `BoFontFamily.inter`
- `whiteSpace` - The white space of the text (optional)
  - default: `BoTextWhiteSpace.nowrap`
- `cssClass` - The CSS class to be applied to the text (optional)
  - default: `undefined`

## Sizes

The `size` prop can be used to change the size of the text. The `size` prop should be a member of the `BoFontSize` enum.

- `extra-small`
- `small`
- `default`
- `body`
- `modal-title`
- `title`
- `h6`
- `h5`
- `h4`
- `h3`
- `h2`
- `h1`
