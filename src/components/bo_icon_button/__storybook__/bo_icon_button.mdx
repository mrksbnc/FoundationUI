# bo-icon-button

An icon only button is a control that allows the user to perform an action.

## Basic usage

```html
<bo-icon-button :icon="Icon.alert_circle" />
```

## Props

- `icon` - The name of the icon (required)
  - Icon name from the `Icon` enum
- `size` - The size of the icon (optional)
  - default: `BoSize.default`
- `variant` - The variant of the icon (optional)
  - default: `BoIconButtonVariant.primary`
- `shape` - The shape of the icon (optional)
  - default: `BoIconButtonShape.default`
- `style` - The style of the icon (optional)
  - default: `BoIconButtonStyle.default`
- `disabled` - Whether the button is disabled (optional)
  - default: `false`
- `isLoading` - Whether the button is loading (optional)
  - default: `false`

## Sizes

The `size` prop can be used to change the size of the icon. The `size` prop should be a member of the `BoSize` enum.

- `large`

## Variants

To change the color of the component, you can use the predifened `BoIconButtonVariant` enum. The `BoIconButtonVariant` enum includes the following variants:

- `primary`
- `secondary`
- `danger`
- `warning`
- `success`
- `dark`
- `purple`
- `teal`
- `link`
- `link_secondary`
- `link_danger`
- `link_warning`
- `link_success`
- `link_dark`
- `link_purple`
- `link_teal`

## Disabled

The `disabled` prop can be used to disable the button. When the button is disabled, it will not respond to user interactions and will appear in a disabled state.

The `isLoading` prop can be used to show a loading spinner. When the spinner is shown, the button will be disabled and will not respond to user interactions.
