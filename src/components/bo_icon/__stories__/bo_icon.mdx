# bo-icon

An icon is a graphical representation of an object or action. The `bo-icon` component is a component that can be used to display icons in various ways.
This library uses the [Feather Icons](https://feathericons.com/) collection of icons.

## Props

- `icon` - The name of the icon (required)
  - an icon name from the `Icon` enum
- `size` - The size of the icon (optional)
  - default: `BoSize.default`
- `color` - The color of the icon in `hex` format (optional)
  - default: `'currentColor'`

## Sizes

The `size` prop can be used to change the size of the icon. The `size` prop should be a member of the `BoSize` enum.

- `extra_small`
- `small`
- `default`
- `large`
- `extra_large`

<Canvas of={BoIconStories.Sizes} />

## Colors

The `color` prop can be used to change the color of the icon. The `color` prop should be a string that represents the color in `hex` format.
The `color` prop can take the values of the `BoColor` enum or any other color value that can be used until it's in valid `hex` format.
